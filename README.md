# 📌 CRUD Overview

| 🧭 Method | 🔗 Endpoint        | 📄 Description        |
|----------|------------------|----------------------|
| `POST`   | `/api/items`      | ➕ Create new item   |
| `GET`    | `/api/items`      | 📖 Get all items     |
| `GET`    | `/api/items/:id`  | 🔍 Get single item   |
| `PUT`    | `/api/items/:id`  | ✏️ Update item       |
| `DELETE` | `/api/items/:id`  | ❌ Delete item       |
